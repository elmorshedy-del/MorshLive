Setup (one-time)

To push routing live for all visitors:

  1. npx wrangler kv namespace create ROUTING_KV — add id to wrangler.toml
  2. npx wrangler secret put ADMIN_TOKEN — use this as your login token
  3. npx wrangler deploy
